  <h2>📌 About</h2>
  <p>LocalChatroom is a fully local chat application supporting multiple accounts, multimedia messages, and user profiles. Experience real-time chatting without relying on external services!</p>

  <!-- Features -->
  <h2>✨ Features</h2>
  <ul>
    <li>🔐 <strong>Login & Registration</strong> – secure accounts for multiple users.</li>
    <li>💬 <strong>Real-time Chat</strong> between logged-in users.</li>
    <li>🖼️ <strong>Images & Videos</strong> sharing in chat.</li>
    <li>🧑‍🎨 <strong>Profile Pictures</strong> for each user.</li>
    <li>💾 <strong>MongoDB</strong> stores all messages and user data reliably.</li>
  </ul>

  <!-- Installation -->
  <h2>🛠️ Installation & Setup</h2>
  <p>To run the project locally:</p>

  <pre><code>
# Clone the repository
git clone https://github.com/devix05/LocalChatroom.git
cd LocalChatroom

# Install dependencies
npm install

# Start MongoDB (local or remote)
# Adjust Mongo URI in your config if needed

# Start the server
npm start

# Open in browser
http://localhost:3000
  </code></pre>
